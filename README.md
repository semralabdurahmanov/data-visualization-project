# Data Visualization Project

This project focuses on visualizing a cleaned sales dataset using Python.
All visualizations were developed in Visual Studio Code and are presented
in a Jupyter Notebook format to display both code and graphical outputs.

## Project Structure

data-visualization-project/
│
├── data/
│ └── llm_cleaned_final.csv
│
├── notebooks/
│ └── visualization_sales_data.ipynb
│
├── README.md


## Tools & Technologies
- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- Visual Studio Code  

## Visualizations
- Distribution of sales amounts  
- Sales performance by city and store  
- Employee sales comparison  
- Time-based sales trends  
- Correlation analysis  

## How to View the Visualizations
Open the notebook located in the `notebooks` directory. All charts are rendered
directly within the notebook and can be viewed on GitHub without running the code.

Note: The dataset included in this repository is publicly available and used
for demonstration purposes only.
